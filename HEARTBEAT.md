# HEARTBEAT.md - 每日检查清单

## 🔔 待办提醒规则

**每天上午 10:00 检查今日待办，如有则发送提醒，并删除/标记已完成。**

提醒格式：
```
📅 今日待办 (YYYY-MM-DD)

1. [待办内容]
   来源: [引用的原始消息]
```

**提醒后操作：**
- 从 MEMORY.md 中删除该待办
- 在 memory/YYYY-MM-DD.md 中标记为 ✅ 已完成
- 避免重复提醒

## 📋 检查清单

- [ ] 检查 memory/YYYY-MM-DD.md 中的今日待办
- [ ] 检查 MEMORY.md 中的待跟进事项
- [ ] 如有待办，发送提醒消息（带上引用的来源）
- [ ] 提醒后从待办列表中删除
- [ ] 检查邮件/日历（可选）

## 📝 示例提醒

```
📅 今日待办 (2026-03-03)

1. 本地生活菜单上预发，回归其他店铺
   来源: [引用的转发消息内容]
```
